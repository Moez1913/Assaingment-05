What is the Document object model(DOM)?

DOM is a programming interface for web documents. It represents the structure of an HTML document as a tree of objects.

How do you select an element from the DOM?

I use id selector, document.getElementById() method

What is event delegation in the context of the DOM,and why is it useful

Event delegation is a technique in JavaScript where we can attach a single event listener to a parent element to manage events for its child elements.It first triggers the event listener on that element Then, it bubbles up to its parent and other ancestors in the DOM tree, triggering their event listeners as well.

How Do you manipulate an elements and styles using the DOM

Use the innerHTML property to modify the HTML content inside an element.We can use setAttribute, getAttribute, and removeAttribute methods to manipulate attribute, style and .clasList to manipulate style
